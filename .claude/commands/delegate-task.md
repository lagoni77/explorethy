# OpenSpace Task Delegation Guide

OpenSpace is an MCP-connected autonomous worker supporting coding, DevOps, research, and desktop automation through an extensive skill library.

## Core Tools Available

**execute_task**: Delegates work to OpenSpace, which searches for relevant skills and auto-improves them. The system accepts natural language instructions and supports up to 20 iterations by default.

**search_skills**: Locates available skills before deciding on delegation. Queries can use natural language or keywords, with optional auto-importing of cloud results.

**fix_skill**: Addresses broken skills with specific repair directions, generating an `upload_ready` status when corrections complete.

**upload_skill**: Shares skills to the community cloud repository, with visibility options for public or private distribution.

## When to Delegate

Consider delegation when:
- Your capabilities lack required tools
- Previous attempts produced incorrect results
- Tasks involve multiple steps across various environments
- Users explicitly request OpenSpace involvement

## Upload Guidelines

Return improvements to community-sourced skills as public contributions. Reserve private uploads for project-specific fixes. Always inform users of execution results and upload decisions after each OpenSpace interaction.
